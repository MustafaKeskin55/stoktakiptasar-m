## 🚀 Kurulum

### Sistem Gereksinimleri
- Node.js 14.0 veya üzeri
- NPM 6.0 veya üzeri
- Modern web tarayıcısı
- İnternet bağlantısı

### Kurulum Adımları
1. Projeyi klonlayın
```bash
git clone [proje-url]
```

2. Bağımlılıkları yükleyin
```bash
npm install
```

3. Geliştirme sunucusunu başlatın
```bash
npm run dev
```

4. Tarayıcıda açın
```
http://localhost:3000
```
